# 🤖 MrBot

Bot Telegram personnel développé en Python sur Android avec Termux.

## ✨ Commandes disponibles

| Commande | Description |
|----------|-------------|
| /start   | Démarre le bot |

## 🛠️ Tech

- Python 3.13
- python-telegram-bot 22.7
- Termux (Android)

## 🚀 Installation

git clone https://github.com/TheRafoow/monbot.git
cd monbot
python -m venv tele-bot
source tele-bot/bin/activate
pip install -r requirements.txt

## ⚙️ Configuration

Crée un fichier `.env` :
TOKEN=ton_token_telegram
